## House Price Prediction using Linear Regression

# Project Overview
This project predicts house prices based on:
* Square Footage
* Number of Bedrooms
* Number of Bathrooms
It uses a Machine Learning algorithm called Linear Regression.

# Technologies Used
* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

 # Algorithm
Linear Regression is used to predict continuous values like house prices.

# How to Run
1. Install dependencies:
pip install numpy pandas matplotlib scikit-learn
2. Run the code:
python main.py

# Output
The model predicts house prices and displays a graph showing the relationship between square footage and price.

# Files Included
* main.py → Python code
* report.txt → Project report
* output.png → Output screenshot
  
# Conclusion
This project demonstrates a basic machine learning workflow using Linear Regression.
